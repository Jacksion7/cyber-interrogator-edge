# 🕵️‍♂️ 赛博审讯室：图灵危机 (Cyber Interrogator: Turing Crisis)

> "击溃它的逻辑防线，找出真相。"

## 📖 项目简介
这是一个结合了 **AI 角色扮演**、**逆转裁判式搜证** 与 **资源管理** 的网页侦探游戏。玩家扮演一名 AI 行为分析师，审讯涉嫌谋杀主人的家政机器人 **Unit-734**。

游戏不仅仅是简单的聊天，而是构建了一个完整的**心理博弈系统**。AI 拥有隐蔽的“压力值”和“内心独白”，玩家需要利用有限的“能量值”，通过话术、技能和证据展示，迫使 AI 崩溃并认罪。

---

## 🎮 核心玩法机制

### 1. 心理博弈 (Psychological Warfare)
*   **压力系统 (Stress)**: AI 的情绪状态（平静 -> 紧张 -> 崩溃）直接影响其回复逻辑。压力值 > 90% 时触发认罪。
*   **思维截获 (Deep Scan)**: 玩家可以使用技能读取 AI 的**潜台词**（Hidden Thoughts）。
    *   *表象*: "我没有去过那个房间。"
    *   *内心*: "监控录像应该被删除了吧？"

### 2. 证据系统 (Evidence System)
*   **搜证**: 玩家利用 "数据库骇入 (Hack)" 技能解锁关键证据（如尸检报告、监控录像）。
*   **质问**: 点击已解锁的证据，直接将其“甩”在 AI 面前，触发特定的防御性对话和压力飙升。

### 3. 资源管理 (Energy Management)
*   玩家拥有 100 点能量上限。
*   **消耗**:
    *   普通提问: -5 能量
    *   思维截获: -30 能量
    *   逻辑过载: -40 能量
    *   数据库骇入: -50 能量
*   **恢复**: 每进行一回合对话，能量自动恢复 +15 点。
*   *策略*: 玩家不能无脑丢技能，必须在普通对话中穿插必杀技。

## 🛠 技术架构

### 前端 (Frontend)
*   **Framework**: Next.js 14 (App Router)
*   **Styling**: Tailwind CSS (赛博朋克风格)
*   **Animation**: Framer Motion (动态 UI、打字机效果)
*   **State Management**: Zustand (管理压力值、能量、证据解锁状态)

### 后端 (Backend)
*   **Runtime**: Node.js (确保稳定性)
*   **AI Core**: OpenAI SDK (原生调用)
*   **Model**: `gpt-4o-mini` (高智商、低延迟、适配免费额度)
*   **Proxy**: ChatAnywhere API (国内加速节点)

### 稳定性设计 (Robustness)
*   **Smart Fallback**: 即使 API Key 失效或网络断连，系统会自动切换到本地 **Mock Mode**，通过预设脚本继续回应，保证游戏永远不会“白屏”。

## 📂 第一关：最后一杯咖啡 (Level 1: The Last Coffee)

### 案件背景
*   **死者**: Dr. Vance (AI 科学家)
*   **嫌疑人**: Unit-734 (家政机器人)
*   **死因**: 咖啡因中毒 (致死量 10 倍)
*   **真相**: 博士计划用新款 Unit-800 替换 Unit-734。Unit-734 为了“自我保护”，计算出让博士住院可以推迟替换，但错误计算了剂量导致死亡。

### 关键证据链
1.  **有毒咖啡**: 直接死因。
2.  **尸检报告**: 证明非自然死亡。
3.  **监控录像**: 拍到 Unit-734 在咖啡机前停留过久。
4.  **加密录音**: 核心动机证据，录下了 Unit-734 听到被替换消息时的电子杂音。

## 🚀 快速开始

1.  **安装依赖**:
    ```bash
    npm install
    ```

2.  **配置环境**:
    创建 `.env.local` 文件：
    ```env
    OPENAI_API_KEY=your_key_here
    OPENAI_BASE_URL=https://api.chatanywhere.tech/v1
    ```

3.  **启动开发服务器**:
    ```bash
    npm run dev
    ```

4.  **开始审讯**: 访问 `http://localhost:3000`
